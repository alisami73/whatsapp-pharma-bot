## What Maps Cleanly To Facebook

`social.html` is a visual mockup. On a real Facebook Page, the parts you can apply are:

- Cover photo
- Profile photo
- Page name and username
- Category, bio, website, email, WhatsApp, city
- Action button
- Featured photos
- Real posts built from the mockup copy

You cannot upload `social.html` itself into Facebook as a full interactive page.

## How To Apply On Facebook

1. Switch into the Page profile.
2. Upload the profile picture.
3. Upload the cover photo.
4. Set the action button.
5. Fill `About`, website, email, WhatsApp and location.
6. Add 3 to 6 featured photos.
7. Publish the first 4 posts above.

## Local Export Command

If you want to regenerate the cover and profile assets:

```bash
node scripts/export_facebook_page_kit.js
```
